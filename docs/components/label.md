# label

Basic label.

```less
@import 'typro/components/label';
```

```html
<span class="label">some label</span>
```


## Modifiers

* `.label--error`, `.label--danger`, `.label--inactive`
* `.label--success`, `.label--active`
* `.label--warning`


## Variables

* `@typro-label-default`
* `@typro-label-warning`
* `@typro-label-danger`
* `@typro-label-success`
