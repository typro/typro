# button

Basic button.

```less
@import 'typro/06-components/button';
```

```html
<a href="#" class="button">My Button</a>
<button class="button">My Button</button>
<input type="submit" value="My Button" class="button">
```


## Modifiers

* `.button--danger`
* `.button--success`
* `.button--small`
* `.button--xsmall`
* `.button--large`
* `.button--xlarge`
